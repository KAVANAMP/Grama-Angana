---

## 📌 About the Project

**Grama-Angana** (meaning "village courtyard" in Kannada) is a mobile solution for managing village or community-level administrative tasks. It brings local governance to the fingertips of residents and administrators through a simple Android app.

---

## ✨ Features

- 👥 Resident/citizen record management
- 📋 Administrative task tracking
- 🗂️ Structured data storage and retrieval
- 📱 Native Android mobile interface
- 🔔 Notifications for community updates

## 🚀 Getting Started

### Prerequisites
- Android Studio (latest version)
- Android SDK
- Java JDK 8 or above
- Android device or emulator (API level 21+)

### Installation

```bash
# Clone the repository
git clone https://github.com/KAVANAMP/Grama-Angana.git
```

1. Open **Android Studio**
2. Click **"Open an existing project"** and select the cloned folder
3. Let Gradle sync complete
4. Click ▶️ **Run** to launch on emulator or connected device
